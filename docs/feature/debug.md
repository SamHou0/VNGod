# 调试

在设置界面中，我们提供了多种调试功能，帮助你进行故障排除。

- 打开日志文件夹。点击这个按钮，就可以打开 VNGod 的日志。默认情况下，只有 Info 以及以上的级别才会写入日志，防止你的硬盘被塞满。当你发现出现问题的时候，可以到 VNGod 的主程序文件夹中，查找 log4net.config，然后把 Info 改成 Debug，这样的日志更加详细。反馈问题时，带上 Debug 级别的日志更加有助于问题的解决。
- 抛出测试异常。这个功能只是为了测试程序崩溃时能否自动提交问题报告，并选择重启。通常情况下，对于用户没用什么用处。
- 运行认证测试。点击此处，可以确定你填入的 Bangumi API Token 是否有效。如果令牌无效或过期，它会报错。此时你应该再去[生成一个](https://next.bgm.tv/demo/access-token/create)。如果没有填入令牌（令牌为空），那么这项检查可以通过，只是你无法访问 NSFW 内容（这是 Bangumi 的规定，只有账号登录且账号达到一定注册时长才能够访问 NSFW 内容）。
