## 功能

- 统一管理你的视觉小说
- 基于 .NET WPF 构建，UI 流畅高效
- 自动匹配在线的条目，支持：
  - [Bangumi](https://bgm.tv)
  - [Vndb](https://vndb.org)
- 快速打开文件夹、游戏和上述条目页面
- 支持提取应用图标显示
- 记录你的游玩时间
- 通过 WebDAV，同步扫描到的数据和游戏存档
- 一键将本地游戏备份到 WebDAV 云端存储（需要调用外部 7-zip 分卷压缩来归档）
- 提供文件管理，可隐藏可删除本地文件夹
